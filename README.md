# LP — Ana Flávia | Nutricionista Esportiva & Estética

Landing page de alta conversão para a Nutri Ana Flávia ([@anamundt](https://instagram.com/anamundt)).

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** com paleta customizada
- **Google Fonts** — Cormorant Garamond + DM Sans

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Adicionar as fotos

Coloque as fotos na pasta `public/fotos/` com os nomes:
- `foto-hero.jpg` — foto principal (Hero)
- `foto-sobre.jpg` — foto da seção Sobre Mim

Veja o arquivo `public/fotos/INSTRUCOES-FOTOS.md` para detalhes.
